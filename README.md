# Machine-Learning-DataSets-Challenge
## 📌 Project Overview

This project is part of my Machine Learning Challenge, where I solve real-world datasets by applying end-to-end machine learning workflows. The goal is to improve my skills in data preprocessing, feature engineering, model building, evaluation, and deployment while tackling practical business problems.

## 📊 Project Workflow

### 1. Data Collection
- Loaded the dataset
- Explored data structure
- Identified data types

### 2. Data Preprocessing
- Missing value treatment
- Duplicate removal
- Outlier detection and handling
- Data type conversion

### 3. Exploratory Data Analysis (EDA)
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Feature Distribution
- Data Visualization

### 4. Feature Engineering
- Encoding categorical variables
- Feature extraction
- Feature selection
- Feature scaling

### 5. Model Building
Implemented and compared multiple machine learning algorithms:

- Linear Regression / Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost
- CatBoost
- Support Vector Machine (Optional)
- Stacking Model

## 📈 Model Evaluation

Evaluation metrics used:

### Regression
- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

### Classification
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
